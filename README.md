## 주요 연구 분야

- **로봇 시스템 구축**: ROS2 기반 통합 제어 시스템 구현
- **최적 경로 생성**: 다자유도 로봇의 효율적인 움직임 계획
- **시스템 아키텍처**: 분산 제어 시스템 설계
- **역학 해석**: 로봇 메커니즘의 역학적 분석
